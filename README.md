# Adria Client
## Quickstart

```bash
git clone <client-repo-url>
cd client
npm install
npm run dev
```


## Naming conventions

| Type       | Naming          | Example                 |
|------------|-----------------|-------------------------|
| Pages      | `[Name]Page`    | `ShopPage`              |
| Components | `[Name]Component` | `SubscriptionComponent` |
